const

发表于 2022-04-18 更新于 2022-05-15 分类于 C++

1. const 含义

常类型是指使用类型修饰符const说明的类型，常类型的变量或对象的值是不能被更新的。

2. const 作用

  • 可以定义常量

  • 类型检查

    • const常量与#define宏定义常量的区别：

      const 常量具有类型，编译器可以进行安全检查；

      #define宏定义没有数据类型，只是简单的字符串替换，不能进行安全检查

    • const 定义的变量只有类型为整数或枚举，且以常量表达式初始化时才能作为常量表达式。

    • 其他情况下它只是一个 const 限定的变量，不要将与常量混淆。

  • 防止修改，起保护作用。

  • 可以节省空间，避免不必要的内存分配
    • const 定义常量从汇编的角度看，只是给出了对应的内存地址，而不是#define一样给出的是立即数。
    • const 定义的常量在程序运行过程中只有一份拷贝。而#define定义的常量在内存中有若干个拷贝。

3. const 对象默认为文件局部变量

注意：非const变量默认为extern。要使const变量能够在其他文件中访问，必须在文件中显式地指定它为extern。

4. 定义常量

上述有两个错误：

  • b 为常量，不可更改！
  • i 为常量，必须进行初始化！(因为常量在定义后就不能被修改，所以定义时必须初始化。)

小结：
