72. 编辑距离

给你两个单词 word1 和 word2， 请返回将 word1 转换成 word2 所使用的最少操作数 。

你可以对一个单词进行如下三种操作：

  • 插入一个字符
  • 删除一个字符
  • 替换一个字符

示例：

输入：word1 = "horse", word2 = "ros"

输出：3

解释：

horse -> rorse (将 'h' 替换为 'r')

rorse -> rose (删除 'r')

rose -> ros (删除 'e')

思路：

if s1[i] == s2[j]:

  • 什么也不做
  • i, j同时向前移动

else:

​ 三选一：

  • 插入（inser）

  • 删除（delete）

  • 替换（replace）

dp(i,j)定义：返回s1[0...i] 和 s2[0...j] 的最小编辑距离
