第二步：点击Profiles驱配置

我打算选用Dracula主题

点击Dracula

否则可以直接下载dracula中给出的.zip文件。

  2. Open the Color Presets... drop-down in the bottom right corner

  3. Select Import... from the list

  4. Select the Dracula.itermcolors file

  5. Select the Dracula from Color Presets...
